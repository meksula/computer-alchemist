Computer alchemist v 1.0 API Documentation

HTTP Method URI Address Parameters Description
POST /set/{username}
  • @param `username` define name of User creating a set.
    		•

    To create empty Computer Set able to assemble. Returned JSON object contain assigned id what you can use to localise your Computer Set.


    Example: /set/John_Doe

    GET /set/{type}/{id}
  • @param `type` define type of Computer Set (Look at list of parameters).
  • @param `id` define ID number in database.
    		•

    You can view any Computer Set what is marked as `public`.


    Example: /set/gaming/53

    GET /set/{type}
  • @param `type` define type of Computer Set (Look at list of parameters).
    		•

    You can view list of Computer Sets of `type` what is marked as `public`.


    Example: /set/work

    PUT /set/{type}/{id}
  • @param `type` define type of Computer Set (Look at list of parameters).
  • @param `id` define ID number in database.
    		•

    The most important part of this module. Here you can assemble Component with Computer Set.
    There is required JSON Object to PUT:
    {"componentType" : "ram", "id" : 3443}
    Above JSON define type and id of Component to be assembled to Computer Set.


    Example: /set/work/344

    DELETE /set/{type}/{id}
  • @param `type` define type of Computer Set (Look at list of parameters).
  • @param `id` define ID number in database.
    		•

    Remove component from database.


    Example: /set/work/344
