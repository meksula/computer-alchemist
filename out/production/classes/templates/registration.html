Create CA account

HTTP Method URI Address Parameters Description
POST /api/v1/registration

To register in CA API execute POST request with JSON (Required JSON format bellow):


{"username" : "...",
"name" : "...",
"surname":"...",
"email" : "..."
"password":"...",
"bornyear" : ... ,
"address" : "{ //optional, if You want use CA Store }" }
If you want to use Computer Alchemist Store (CAS) from one account, please enter your address too. You can do it later of course.

POST /api/v1/login

Use this endpoint to login to CA API


(Required JSON format):
{"username" : "...", "password" : "..."}
